# TripSync

# Back-end

- Lệnh cài đặt venv: python -m venv venv

- Lệnh khởi động venv: source venv/Scripts/activate hoặc venv\Scripts\Activate.ps1 (lưu ý trỏ đúng thư mục chứa venv)

- Lệnh cài đặt fastapi: pip install fastapi uvicorn

- Lệnh cài đặt package trong file requirement.txt: pip install -r requirements.txt

- Lệnh check kết nối DB: python -m app.database

- Lệnh chạy fastapi : uvicorn app.main:app --reload
